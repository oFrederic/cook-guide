# Styles

This directory contains global styles and theme customization.

## Structure

- `theme/` - Chakra UI theme customization
- `global/` - Global CSS styles
- `animations/` - CSS animations and transitions

## Usage Guidelines

- Prefer Chakra UI's styling system over custom CSS when possible
- Use CSS modules for component-specific styles
- Define reusable theme tokens in the theme configuration
- Keep global styles minimal
