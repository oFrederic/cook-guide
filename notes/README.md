# CookGuide App

This directory contains the essential information about the CookGuide App project - a detailed recipe website that serves as both a portfolio piece and a useful daily tool.

## Purpose of This Directory

These files help track:
- The project overview and vision
- Technology stack and architecture decisions
- Development roadmap and progress
- Key features and implementation details

## Files to Reference

When working on this project, review:

- **PROJECT.md** - Contains the project overview, technology stack, and architecture
- **ROADMAP.md** - The detailed development plan with all phases and tasks
- **README.md** - This file, with general guidance

## Using These Files

When starting a new development session:
1. Review `PROJECT.md` to recall the project's purpose and technology choices
2. Check `ROADMAP.md` to see which tasks are active and which are completed
3. Update these files as you make progress to maintain continuity

When completing tasks:
1. Update `ROADMAP.md` to mark completed tasks with `[x]` instead of `[ ]`
2. Include a reference to the ROADMAP.md update in your commit message
