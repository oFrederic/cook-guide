# Utility Libraries

This directory contains utility functions and shared logic.

## Structure

- `utils/` - General utility functions
- `api/` - API-related functions and services
- `helpers/` - Helper functions for specific features
- `constants/` - Application constants and configuration

## Usage Guidelines

- Functions should be pure and well-documented
- Use TypeScript for type safety
- Write tests for utility functions
- Group related utilities in appropriate subfolders
