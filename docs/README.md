## 📋 Project Quick Reference

| Aspect | Details |
|--------|---------|
| **Project Type** | Full-stack recipe website with interactive features |
| **Frontend** | Vue 3 + TypeScript + Vuetify 3 + Vite |
| **Backend** | Express.js + TypeScript + Prisma + PostgreSQL |
| **Authentication** | JWT + Passport.js + bcryptjs |
| **Image Storage** | Cloudinary |
| **Testing** | Vitest + Vue Test Utils + Cypress + Supertest |
| **Deployment** | Vercel/Netlify (frontend) + Railway (backend) |
