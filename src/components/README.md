# Components

This directory contains all reusable UI components.

## Structure

- `ui/` - Basic UI components (buttons, inputs, cards, etc.)
- `layout/` - Layout components (header, footer, sidebar, etc.)
- `features/` - Feature-specific components
- `forms/` - Form components and form-related elements

## Usage Guidelines

- Components should be as reusable as possible
- Use TypeScript interfaces for props
- Create subfolders to organize related components
- Include appropriate tests for each component
