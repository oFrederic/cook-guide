# App Router

This directory contains all pages and layouts for the Next.js App Router.

## Structure

- `layout.tsx` - Root layout component
- `page.tsx` - Home page component
- `(routes)/` - Route groups for application sections

## Usage Guidelines

- Use Server Components by default
- Add 'use client' directive only when necessary
- Organize routes based on URL structure
- Use route groups (parentheses folders) to organize without affecting URL structure
